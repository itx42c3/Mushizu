# 虫唾

Win+R に「虫唾」と入力すると画像が画面を横切るだけのアプリ。

## ダウンロード

[Releases](../../releases) から最新版の zip をダウンロードしてください。

## 実行

1. zip を解凍して好きな場所に置く
2. `虫唾.exe`を実行
3. `虫唾.exe`をPATH登録

3の作業は必須ではありませんが、3の作業をすることでWin+Rに「虫唾」と入力して実行できます。

## 動作環境

- Windows 10 / 11
- .NET 6.0 以上 ([ダウンロード](https://dotnet.microsoft.com/download/dotnet/6.0))
